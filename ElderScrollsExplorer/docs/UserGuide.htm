Elder Scrolls Explorer User Guide

 

First Use

Setting game media

If this is the first time you’ve run Elder Scrolls Explorer you will need to set the game media folders. This is the location of the *.esm and *.bsa file for the particular game.

If you own the games these will either be in a custom install location, or if you purchased them on steam somewhere like

F:\Games\Steam\steamapps\common\Fallout 3 goty\Data

 

You can set these by going to File->Set Folders

This dialog will pop up (along with this help file) automatically if you have no folders set.

To set folders click the “…” button to the right of each game setting row.

If you’ve been given a password you can also click the “FTP” button next to each game folder setting row and it will download the media and set it automatically, the media can be many GB in size, so only download if you have to.

 

Note that Hunter Sneaker is a custom game whose media can only be acquired through the FTP button.

 

Resolution

When asked for a resolution accepting the default should be fine.

·         Loading cells takes time, there is no wait cursor or progress bar, clicking randomly  will not speed things up (but will cause problems)

·         You can go through doorways, it will show light grey if it's possible but too far away and white when it is possible

·         Use left click to go through the doorway, once again there is no progress bar so just wait

·         You can also operate regular doors and gates by left clicking

 

How to use

Click the game type you want to run

·         Note that once a game type is selected the application needs to be restarted to switch to another

 

After a moment all available cells will be displayed in the table.

Click a row in the table and the selected cell will be loaded and displayed

You can then click another cell and it will be loaded

Your location is updated if you use doors, but simply click a cell will not guarantee the cells contents are where you are. Interior cells tend to be around x=0, y=0, z=0 exterior cells can be anywhere, so you might see nothing interesting if you location is away from the content.

 

 

 

Notes

You must have the original game *.esm and *.bsa files. This requires that you have rights to the media files of those games.

If you adjust position via the Quick Edit Location, note that

·         Y is +up  -down

·         X is +east –west

·         Z is –north +south

Yes I know Z is crazy, it’s a historic thing

 

A lot of the graphics settings are not working so playing with them may or may not do anything.
